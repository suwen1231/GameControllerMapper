# 🎮 GameControllerMapper

一款強大的開源工具，旨在將 **遊戲手把（Xbox、PS4、Switch）** 轉換為高效的 **生產力輔助設備**。透過自定義按鍵映射，您可以利用手把來操作繪圖軟體、影片剪輯工具或任何需要頻繁使用快捷鍵的應用程式 

---

## 🚀 快速開始

### 1. 下載與執行
* 下載軟體壓縮包並解壓縮。
* 在資料夾內點選 `GameControllerMapper.exe` 本工具就會啟動。

### 2. 重要權限說明
> [!IMPORTANT]
> **建議「以系統管理員身分執行」**
> 由於 Windows 的安全機制，若目標軟體權限高於本工具，模擬的訊號將會失效。使用管理員權限執行可確保訊號在所有軟體中（如工作管理員）都能運作。

## ✨ 核心功能

* **多平台手把支援：** 支援 Xbox、PS4 與 Switch 的視覺化按鍵配置與符號切換。
* **強大的組合鍵 (Combo) 功能：**
* **預設組合鍵：** 內建常用指令如「複製 (Ctrl+C)」、「截圖 (Win+Shift+S)」等。
* **自定義組合鍵：** 支援設定多達 **五個按鍵** 的組合，可穩定觸發複雜快捷鍵。
* **滑鼠模擬：** 可將搖桿映射為滑鼠移動（支援靈敏度調整），按鈕可映射為左鍵、中鍵、右鍵或滾輪。
* **視覺化回饋：** 運作時，手把圖示上方會顯示 **藍色懸浮 Badge**，即時反饋按鍵功能與按下狀態。
* **設定檔管理：** 支援保存與讀取 `.xmap` 設定檔。
